  БЫСТРАЯ КОНВЕРТАЦИЯ В APK (5 минут)
═══════════════════════════════════════════════════════════════

ШАГ 1: Загрузите приложение в интернет
───────────────────────────────────────────────────────────────
1. Откройте: https://app.netlify.com/drop
2. Перетащите ВСЮ папку приложения в браузер
3. Дождитесь загрузки (получите ссылку типа: 
   https://random-name-123.netlify.app)
4. Скопируйте эту ссылку


ШАГ 2: Конвертируйте в APK
───────────────────────────────────────────────────────────────
1. Откройте: https://www.pwabuilder.com
2. Вставьте ссылку из шага 1
3. Нажмите "Start"
4. После проверки нажмите "Build My PWA"
5. Выберите "Android" → "Generate Package"
6. Заполните:
   - Package ID: com.spe.tickets
   - App Name: СПЭ Билеты
   - Version: 1.0.0
7. Нажмите "Generate APK"
8. Скачайте готовый APK


ШАГ 3: Установите на телефон
───────────────────────────────────────────────────────────────
1. Передайте APK файл на Android телефон
2. В настройках включите "Установка из 
   неизвестных источников"
3. Откройте APK файл
4. Нажмите "Установить"
5. Готово! Приложение установлено


⚠ ВАЖНО:
- Приложение должно быть доступно по HTTPS
- Все файлы должны быть загружены
- Service Worker должен работать


Подробная инструкция в файле: PWA_TO_APK_ИНСТРУКЦИЯ.md
